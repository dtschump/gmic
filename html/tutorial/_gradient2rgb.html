-gradient2rgb

Figure 1: Blurring map by way of -gradient2rgb 1

This command internally computes 2D direction or 2D orientation fields of the selected images and produces a color representation of the fields.

The command has the following format:

-gradient2rgb: _is_orientation={ 0 | 1 }

  1. orientaton: Chooses a 2D direction field, 0, the default, produced via -gradient, or a 2D orientation field, 1, produced by -gradient_orientation.

Frequently, one creates a 2D direction or orientation field and then produce a color representation via -direction2rgb. This command consolidates this common idiom into one command.

This command is a handy source of blurring maps which trace contours for the G'MIC-Gimp Blur by Color filter.
